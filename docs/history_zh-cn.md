# 📦 历史版本

2. 内置Minecraft方块、物品、附魔、药水等全套ID数据库（`data.json`）
3. 关键词+分类多条件筛选检索功能

5. 一键复制 ID / 完整指令到剪贴板，操作提示弹窗
6. 完善文件缺失、解析失败等异常捕获弹窗提示
7. 新增配套文档

# 2026-7-7 v1.1.0 Beta1
1. 更新了深色模式（Beta）
